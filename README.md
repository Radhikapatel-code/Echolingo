# Echolingo

A Python script to transcribe, translate, and dub videos (e.g., Hindi to English) using Whisper, Google Translate, gTTS, and moviepy.

## Features
- Outputs a video with dubbed audio and subtitles from one language to another.

## Requirements
- Libraries: `openai-whisper`, `moviepy==1.0.3`, `googletrans==3.1.0a0`, `gtts`, `srt`, `torch`
- FFmpeg 7.1.1
- ImageMagick (with legacy utilities)
